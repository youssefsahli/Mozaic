# API Specification

This project now includes an autogenerated API specification generated from TypeScript sources.

## Output

- Machine-readable spec: [docs/api-spec.json](docs/api-spec.json)

## Generate / Refresh

```bash
npm run docs:api
```

## Watch Mode

```bash
npm run docs:api:watch
```

## Source Coverage

The API spec is generated from:
- `src/editor/**`
- `src/engine/**`
- `src/parser/**`
- `src/index.ts`

## Notes

- Test files are excluded.
- The spec is generated via TypeDoc and intended to be committed so reviewers can diff API changes.
